DOM Animaciones

La animación en el DOM se logra al cambiar gradualmente las propiedades de los elementos HTML y aplicar transiciones suaves entre diferentes estados. Esto puede incluir cambios en la posición, tamaño, color y otros atributos visuales de los elementos.

La animación en el DOM se realiza mediante el uso de métodos y propiedades proporcionados por JavaScript. Algunas de las técnicas comunes para crear animaciones incluyen el uso de la función setInterval() o requestAnimationFrame() para actualizar continuamente las propiedades de los elementos y el uso de transiciones CSS para lograr efectos de interpolación suave.

Las animaciones DOM son ampliamente utilizadas en el desarrollo web para crear interacciones y efectos visuales atractivos. Al aprovechar el poder del DOM y JavaScript, es posible crear animaciones dinámicas y responsivas que mejoren la experiencia del usuario en un sitio web.

Ejemplo 1

function miAnimacion() {
    let id = null;
    const elem = document.getElementById("animate");
    let pos = 0;
    clearInterval(id);
    id = setInterval(frame, 5);
    function frame() {
        if (pos == 75) {
            clearInterval(id);
        } else {
            pos++;
            elem.style.top = pos + 'px';
            elem.style.left = pos + 'px';
        }
    }
}

DOM Eventos

DOM Events se refiere a los eventos que ocurren en un documento HTML y que pueden ser detectados y controlados mediante JavaScript. Los eventos son acciones que suceden en la página web, como hacer clic en un elemento, mover el mouse, pulsar una tecla del teclado, cargar una página, etc.

El DOM (Document Object Model) proporciona una interfaz para trabajar con los eventos en el navegador. Cuando ocurre un evento, se genera un objeto de evento que contiene información sobre ese evento específico, como el tipo de evento, el elemento objetivo, las coordenadas del mouse, las teclas presionadas, entre otros detalles.

Mediante el uso de JavaScript, es posible asignar manejadores de eventos a los elementos HTML para responder y controlar las acciones del usuario. Los manejadores de eventos son funciones que se ejecutan cuando un evento en particular ocurre en un elemento específico.

Los eventos del DOM permiten crear interactividad y responder a la interacción del usuario en una página web. Al detectar y manejar eventos, es posible realizar acciones personalizadas, validar formularios, actualizar elementos en tiempo real, realizar animaciones, entre muchas otras posibilidades.

En resumen, los DOM Events son eventos que ocurren en un documento HTML y pueden ser detectados y controlados mediante JavaScript para crear una experiencia interactiva y dinámica en una página web.

DOM Event Listener

El método addEventListener() es una función utilizada en JavaScript para asignar un evento a un elemento HTML específico, como un botón, un enlace o cualquier otro elemento interactivo en una página web. Permite que el código JavaScript responda a eventos del usuario, como hacer clic en un elemento, pasar el cursor sobre él o enviar un formulario.

La sintaxis básica de addEventListener() es la siguiente:

elemento.addEventListener(evento, función, useCapture);
  • elemento: es el elemento HTML al que se le asignará el evento.
  • evento: es una cadena de texto que especifica el tipo de evento que se va a manejar, como "click", "mouseover" o "submit".
  • funcion: es la función de JavaScript que se ejecutará cuando ocurra el evento.
  • useCapture (opcional): es un valor booleano que indica si el evento se captura durante la fase de captura (true) o durante la fase de bubbling (false). Generalmente, se usa false para la mayoría de los casos.

Cuando se llama a addEventListener(), la función especificada se ejecutará cuando ocurra el evento en el elemento indicado. Puedes utilizar addEventListener() para asociar múltiples eventos a un mismo elemento o para asignar diferentes eventos a diferentes elementos.

Aquí hay un ejemplo de cómo usar addEventListener() para capturar el evento de clic en un botón:

Ejemplo 1

function adel_click(){
    document.getElementById("miTexto8").innerHTML = "Se ha reconocido el evento click! :D";
}
document.getElementById("miBoton").addEventListener("click",adel_click);

DOM Node

El HTML DOM le permite cambiar el contenido HTML utilizando la propiedad "innerHTML". El "innerHTML" se usa típicamente en páginas web para generar contenido HTML dinámico, como formularios de comentarios, formularios de registro y enlaces.

Sintaxis: elemento.innerHTML = valor;

Ejemplo

function displayPhrase()
{
    document.getElementById("p1").innerHTML = "Este texto fue remplazado";

var items = document.querySelectorAll(".item2");
var btnCambiarEstilo = document.getElementById("btnCambiarEstilo");
var btnCambiarTamaño = document.getElementById("btnCambiarTamaño");
var colorOriginal = true;
var tamañoOriginal = true;

btnCambiarEstilo.addEventListener("click", function() {
    if (colorOriginal) {
    items.forEach(function(item) {
        item.style.color = "red";
    });
    colorOriginal = false;
    } else {
    items.forEach(function(item) {
        item.style.color = "";
    });
    colorOriginal = true;
    }
});

btnCambiarTamaño.addEventListener("click", function() {
    if (tamañoOriginal) {
    items.forEach(function(item) {
        item.style.fontSize = "24px";
    });
    tamañoOriginal = false;
    } else {
    items.forEach(function(item) {
        item.style.fontSize = "";
    });
    tamañoOriginal = true;
    }
});
